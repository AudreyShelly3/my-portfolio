# my-portfolio

This repository contains my personal portfolio website where I showcase my projects, technical skills, and background in Information Technology and cybersecurity.

The website was built as part of my learning journey in web development and serves as a central place to present my work and experience.

---

## Overview

The portfolio website introduces who I am, my technical interests, and the projects I have developed during my IT studies. It also provides visitors with a way to explore my GitHub repositories and learn more about my work.

---

## Features

• Responsive design for desktop and mobile devices  
• About section introducing my background  
• Skills section highlighting technologies I work with  
• Project cards displaying my completed work  
• Interactive project details using JavaScript  
• Contact section for communication
